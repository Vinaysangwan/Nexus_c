# Nexus_c
  * raylib/ SDL3 like library for graphics rendering
  * Users have full access of funcitons upto OpenGL 4.3

## To Use the lib you only need 2 Things:
  * Nexus_c.h (inside the include folder)
  * Nexus_c.lib (PreBuilt one / You can built for youself)

## Compatiable Devices:
  * **Windows**
  * **Mac** (Work on Progress)
